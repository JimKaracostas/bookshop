This is a comprehensive Bookshop Management System developed in Python. It features a graphical user interface (GUI) using Tkinter and a robust Model-View-Controller (MVC) architecture to manage bookstore inventory, sales, and databases.

## ✨ Features

- **Inventory Management:** Add, edit, and remove books from the store.
- **Sales Tracking:** Keep track of sold books and generate reports.
- **Graphical Interface:** Built completely with Tkinter for an easy-to-use desktop experience.
- **Database Integration:** Securely stores all book and transaction records locally.
- **MVC Architecture:** Clean codebase separating the models, views (pages), and controllers.

## 🚀 Getting Started

### Prerequisites
* Python 3.x installed on your system.
* Required Python libraries (e.g. `tkinter`).

### Running the application
To start the Bookshop Management System, simply run `main.py` from your terminal:

```bash
python main.py
```

## 🛠️ Project Structure

- `main.py` - Application entry point.
- `controllers/` - Contains the logic connecting the UI to the database.
- `models/` - Data structures and business logic.
- `pages/` - UI views built with Tkinter.
- `db.py` - Database configuration and operations.
